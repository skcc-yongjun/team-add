# IES 평가 통계 화면 (IES-SCR-007)

## 사용 역할자: 시스템 관리자 (IES), 입학처 담당자

---

## 기능 1: 면접·실기 평가 통계 조회

-   **관련 요구사항:** IES-REQ-015: 면접·실기 평가 통계 및 보고서 생성
-   **입력 정보:**
    -   조회할 통계 유형 선택 (평가 항목별 평균/분포, 평가위원별 평균/편차, 모집 단위별/조별 비교 등).
    -   조회 조건 설정 (모집 시기, 전형, 고사일 등).
    -   '통계 조회' 버튼 클릭.
-   **출력 정보:**
    -   선택한 통계 데이터가 화면에 차트(막대, 선, 분포 곡선 등) 및 테이블 형태로 시각화되어 표시됨.
    -   예시:
        -   평가 항목별 점수 분포 히스토그램.
        -   평가위원 간 평균 점수 비교 막대 차트.
        -   시간대별 평가 진행률 현황.
-   **프로세스 설명:**
    1.  사용자가 필요한 통계 유형과 조건을 선택한다.
    2.  '통계 조회' 버튼 클릭 시, 클라이언트는 해당 요청을 API로 전송한다.
    3.  서버는 DB의 `EVALUATION` 테이블 등에서 평가 결과 데이터를 집계/분석하여 통계치를 계산한다.
    4.  계산된 통계 데이터를 클라이언트에 반환한다.
    5.  클라이언트는 수신된 데이터를 사용하여 차트 라이브러리 등으로 시각화하여 화면에 표시한다.
-   **호출 API 정보:**
    -   HTTP 메서드: GET
    -   API 엔드포인트: `/api/ies/statistics/{statisticType}` (예: `/api/ies/statistics/item-distribution`)
    -   API 목적: 특정 유형의 평가 통계 데이터 조회
    -   요청 파라미터: 조회 조건 (`admissionRound`, `admissionType`, etc.)
    -   응답 파라미터: 통계 데이터 (JSON 형식)
-   **Mockup 데이터 예시:**
    -   입력: '평가위원별 평균 점수' 통계, '2024 수시 학생부종합' 조건으로 조회.
    -   출력: 각 평가위원의 평균 점수를 보여주는 막대 차트와 테이블 표시.

## 기능 2: 평가 통계 보고서 생성 및 다운로드

-   **관련 요구사항:** IES-REQ-015: 면접·실기 평가 통계 및 보고서 생성
-   **입력 정보:**
    -   생성할 보고서 종류 선택 (기능 1의 통계 유형과 연동 또는 별도 정의된 보고서).
    -   보고서 대상 조건 설정.
    -   '보고서 생성/다운로드' 버튼 클릭.
-   **출력 정보:** 선택한 통계 보고서가 파일(Excel, PDF 등)로 생성되어 다운로드됨.
-   **프로세스 설명:** 기능 1에서 조회된 통계 또는 미리 정의된 형식의 보고서를 파일로 생성하는 기능. 서버는 통계 데이터를 계산하고 이를 보고서 템플릿에 적용하여 파일을 생성 후 반환한다.
-   **호출 API 정보:**
    -   HTTP 메서드: GET 또는 POST
    -   API 엔드포인트: `/api/ies/statistics/reports/{reportType}`
    -   API 목적: 평가 통계 보고서 파일 생성 및 다운로드
    -   요청 파라미터: 조회 조건
    -   응답 파라미터: 파일 스트림
-   **Mockup 데이터:** 없음 (파일 다운로드)
